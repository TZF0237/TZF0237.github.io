## LaTeX基础学习

<!-- more -->

## 1. LaTeX的基本结构

### 1.1 基本结构

LaTeX 中主要分为导言区和正文区。

```
% 导言区
\documentclass{article}
\usepackage{ctex}

%正文区
\begin{document}
    Hello
\end{document}
```

### 1.2 导言区

导言区通常用于定义文档的格式、语言等（全局设置）。

#### \documentclass

|  Class   |            简介            |
| :------: | :------------------------: |
| article  | 科技论文，报告，软件文档等 |
| IEEEtran |   IEEE Transactions 格式   |
|  report  |     长篇报告，学位论文     |
|   book   |            书籍            |
|  beamer  |            PPT             |

|        Option        |                  简介                  |
| :------------------: | :------------------------------------: |
|         12pt         |      文档正文字体大小（默认10pt）      |
| a4paper、letterpaper | 页面规格 |

### 1.3 正文区

在正文区，我们可以用公式、用命令构成文本文档。一个 LaTeX 文件中只能有一个 document。

## 2. LaTeX的中文处理办法

如果是 XeLaTeX 或 LuaLaTeX，可以使用 `xeCJK` 宏包；如果是 pdfLaTeX，可以使用 `CJKutf8` 宏包。推荐使用 `ctex` 宏包，最方便。

## 3. LaTeX的篇章结构

使用 `\section`、`\subsection`、`\subsubsection` 构建层级结构，使用 `\tableofcontents` 生成目录。

## 4. LaTeX中的特殊字符

空白字符、控制符（`\# \$ \% \{\} \~{} \_{} \^{} \textbackslash \&`）、排版符号、引号等。

## 5. LaTeX中的插图

引入 `graphicx` 宏包，使用 `\includegraphics[选项]{文件名}` 语法。支持 EPS、PDF、PNG、JPEG、BMP 格式。

## 6. LaTeX中的表格

```latex
\begin{tabular}{l|c|p{1.5cm}|c|r}
\hline
姓名 & 语文 & 数学 & 外语 & 备注 \\
\hline
张三 & 98 & 96 & 97 & 优秀 \\
\hline
\end{tabular}
```

## 7. LaTeX中的浮动体

使用 `figure` 和 `table` 环境，常用浮动格式 `[htbp]`。

## 8. LaTeX中的数学公式

包括行内公式（`$...$`）、行间公式（`$$...$$` 或 `\[...\]`）、上下标、希腊字母、分式、矩阵、多行公式等。

## 9. 参考文献BibTex

创建 `.bib` 文件，使用 `\bibliographystyle{plain}` 指定样式，`\bibliography{文件名}` 生成参考文献。
