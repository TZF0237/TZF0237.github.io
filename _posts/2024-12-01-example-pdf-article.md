## About This PDF Post

This is an example of a **PDF-type blog post**. The actual content is the PDF file embedded above.

### How It Works

1. Upload your PDF file to the `files/` directory (e.g., `files/my-article.pdf`)
2. Create a Markdown file in `_posts/` with the naming convention: `YYYY-MM-DD-title.md`
3. In the front matter, add `pdf_file: "my-article.pdf"`
4. Optionally add `pdf_description` for an explanation below the viewer

### When to Use PDF Posts

- Publishing research papers or preprints
- Sharing slide decks from presentations
- Distributing reports or documentation that should preserve exact formatting
- Archiving content that was originally created in LaTeX or other non-Markdown tools
