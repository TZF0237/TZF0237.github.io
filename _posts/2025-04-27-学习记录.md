### A-while

```c++
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
int main(){
    string s,str="while";
    int ans=0;
    cin>>s;
    for(int i=0;i<5;i++){
        if(s[i]!=str[i]) ans++;
    }
    cout<<ans;
    return 0;
}
```

### B-token

前缀和。

```c++
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
ll n,m,ans=-1,a[100010];
int main(){
    cin>>n;
    for(int i=1;i<=n;i++) cin>>a[i];
    for(int i=1;i<=n;i++){
        m+=a[i];
        if(i-10>0) m-=a[i-10];
        ans=max(ans,m);
    }
    cout<<ans;
    return 0;
}
```

### E-小苯的矩阵反转

题意是仅选择两行或两列或一行一列。讨论四种情况最后形成的图案：

- 什么也不做: 个数为0的行数为n
- 选择两行: 个数为0的行数为n-2, 个数为m的行数为2
- 选择两列: 个数为0的列数为m-2, 个数为n的行数为2
- 选择一行一列: 个数为1的行数为n-1, 个数为m-1的行数为1

### F-小苯的因子查询

假设一个数 num，有 odd 个奇质因数和 even 个偶质因数。想得到奇因数，不能选择 even。偶质因数只有 2，`fac[i]` 代表 i 的阶乘含有质因数 2 的数目，概率就是 `1/(fac[n]+1)`，结合费马小定理求逆元。
